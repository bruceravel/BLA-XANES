Demeter::UI::Metis¶

NAME¶

Demeter::UI::Metis - BLA data processing

VERSION¶

This documentation refers to Xray::BLA version 2.

DESCRIPTION¶

Metis is a graphical interface the Xray::BLA package for processing data from an energy dispersive bent Laue analyzer spectrometer in which the signal is dispersed onto the face of a Pilatus camera.

DEPENDENCIES¶

Xray::BLA and Metis dependencies are in the Build.PL file.
