The Configuration tool¶

METIS uses the same configuration infrastructure as the other DEMETER GUIs. If you have used the configuration tool in ATHENA, ARTEMIS, or HEPHAESTUS, then you know how to use this one.

METIS's configuration tool.

Here links to the configuration documentation in Athena and Artemis. Here is the explanation from the Demeter Programming Guide.

The configuration parameters will be saved in the same demeter.ini file that is used by the other DEMETER programs. On linux (and other unixes) this is in $HOME/.horae/. On Windows this is in %APPDATA%\\demeter.

Not all of the Gnuplot options are used by METIS. However, the parameters ♦gnuplot→col0 through ♦gnuplot→col9 are used as the colors for line plots.
