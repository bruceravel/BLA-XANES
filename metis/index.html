The Metis program¶

The METIS program is modal, providing slightly different functionality for HERFD, XES, and RXES data sets or for individual images. At the command line, it is invoked with a command line argument to put it into the correct mode.

Mode purpose
xes non-resonant emission spectroscopy
herfd high resolution XANES
rxes RIXS plane
mask process an individual elastic image

To start METIS in XES mode, provide the xes argument:

metis xes

METIS, at startup, invoked in XES mode

Similarly, to begin METIS in hrfd, rxes, or mask mode, do

metis herfd
metis rxes
metis mask

Menu bar¶

The Metis menu.

The Metis menu allows you to import and export project save files and provides alternative ways – including keyboard shortcuts – to move between the tools listed in the toolbar. There is also an option for importing configuration information from a configuration file used by the command line program.

The METIS project save file is a normal HDF5 file with the .mpj file extension. It can be read using any HDF5-sware tool. It contains some simple semantics to make it recognizable to METIS.

The Help menu allows you to probe the state of the current analysis project.

The Help menu.

The first item in the menu presents a simple text display showing the state of the Perl object used to organize the analysis. The second item shows a history of status bar messages. The third item shows the Gnuplot commands for the most recently made plot.
